🚀 Mini WeasyPrint 테스트 문서

📋 개요

이것은 Mini WeasyPrint의 기능을 테스트하는 문서입니다. 원래 WeasyPrint와 비교해서 훨씬 가벼운 버전입니다.

주요 특징:

• 단일 파일로 구성

• 최소한의 의존성 (reportlab만 필요)

• 기본적인 HTML/CSS 지원

• 윈도우 환경 최적화

🎨 스타일 테스트

이 텍스트는 일반 단락입니다.

이 텍스트는 하이라이트됩니다.

코드 예제

python mini_weasyprint.py input.html output.pdf

📊 기능 비교

원본 WeasyPrint vs Mini WeasyPrint:

원본 WeasyPrint:

• 8개의 주요 의존성

• 완전한 CSS 지원

• 복잡한 레이아웃

• 큰 설치 용량

Mini WeasyPrint:

• 1개의 주요 의존성 (reportlab)

• 기본적인 CSS 지원

• 간단한 문서 레이아웃

• 작은 설치 용량

✅ 결론

Mini WeasyPrint는 간단한 HTML 문서를 PDF로 변환하는 데 완벽한 솔루션입니다!
